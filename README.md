# Homebrew Tap

Homebrew formulae and casks for [sasha-computer](https://github.com/sasha-computer) projects.

## Install

```bash
brew tap sasha-computer/tap
```

## Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| `domain-search` | Search for available domains across every TLD | `brew install sasha-computer/tap/domain-search` |

## Casks

| Cask | Description | Install |
|------|-------------|---------|
| `claude-code-usage` | See your Claude Code rate limits in the macOS menu bar | `brew install --cask sasha-computer/tap/claude-code-usage` |
