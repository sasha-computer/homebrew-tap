# Homebrew Tap

Homebrew casks for [sasha-computer](https://github.com/sasha-computer) projects.

## Install

```bash
brew tap sasha-computer/tap
brew install --cask claude-code-usage
```

## Casks

| Cask | Description |
|------|-------------|
| `claude-code-usage` | See your Claude Code rate limits in the macOS menu bar |
